# Mavenについて
下記を行うビルドツール
- Javaプロジェクトの構造管理
- 異存関係の管理
- ビルド
- デプロイ

設定ファイル(pom.xml)を使用してプロジェクトの設定を一元管理する
Project Object Model

## 依存関係の管理
1. pom.xmlに依存ライブラリの情報を記述する
2. 1で記載した依存関係に従って、インターネット上に存在するセントラルリポジトリ（Maven Central）にアクセスする
3. セントラルリポジトリからjarファイル（部品をまとめたファイル）をプロジェクト内でMavenが管理する領域（ローカルリポジトリ）にダウンロード

SpringBootはローカルリポジトリに存在する部品をりようしてプログラムを動作させる
