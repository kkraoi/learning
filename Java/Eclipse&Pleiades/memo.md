### 初期設定
1. ワークスペース名
    - 作業を行う空間に名前を付けたもの
    - 実際はフォルダ
    - やり方：
        - 起動時にウィザード(Eclipse IDE ランチャー)が表示され、任意の名称
2. パースペクティブ
    - 各役割のウィンドウ（ビュー）の表示や配置のこと
    - やり方:
        - 右上の「＋」のアイコンを押下。
        - 青いJマークのJava(※デフォルトではない)を開く
        - java開発のセットが展開される
3. テーマ
    - 画面の色設定
    - やり方
        - ウィンドウ > 設定 > 一般 > 外観
        - 設定によってはeclipseの再起動が必要
4. Javaコンパイラのバージョン
    - 使用するJavaコンパイラのバージョン
    - 21を使用するやり方：
        - ウィンドウ > 設定 > Java > コンパイラー 
        - コンパイラー準拠レベル を 21にする
5. JREのバージョン
    - Java Runtime Environment
    - Javaで開発されたソフトウェアやアプリケーションをパソコンやスマホ上で動かすために必要なソフトウェアのセット
    - JDKとはどう違う？: JDKは「Javaプログラムを開発・実行するためのキット」で、JREは「Javaプログラムを実行するためだけの環境」
    - 21を使用するやり方
        - ウィンドウ > 設定 > Java > インストール済みのJRE
        - JAVA21を選択
6. エンコード
    - テキストファイルに対するエンコード
    - UTF-8を使うやり方
        - ウィンドウ > 設定 > 一般 > ワークスペース
        - テキスト・ファイル・エンコード を UTF-8 に設定
        - 新規テキスト・ファイルの行区切り文字を Windows に設定
7. テキスト・フォント
    - ウィンドウ > 設定 > 一般 > 外観 > 色とフォント
    - フォントテキスト > テキストフォントをダブルクリック
    - フォントファミリーやサイズを選べる

## プロジェクトを新規作成
- ファイル > 新規 > 新規 Spring スターター・プロジェクト でダイアログを展開
- Eclipseの新規Springスターター・プロジェクト依存関係ダイアログ
    - Spring Boot DevTools
        - プログラムや設定の変更を検知して実行環境に反映したり、デバッグのサポートを行う
    - Spring Web
        - Webアプリをつくるときの枠組みを提供
        - Webアプリを作るときに必須
    - Thymeleaf

## インポートのショートカット
Cntrol + Shift + o

ちなみに、ファイル保存でインポートしてくれる。

## Javadocの生成方法
1. プロジェクト > Javadocの生成
2. プロジェクトを選択
3. 可視性を選択
    - Private: すべてを対象とする
4. 標準ドックレットを使用にチェック
    - 標準ドックレット：Javadocの標準的な書き方
5. 宛先に出力先を設定
