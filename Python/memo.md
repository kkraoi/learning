## 「:」
基本的には、a:b => a以上b未満だが、<br>
`1:2`は2を含む例外がある。

## 「~」
~ は 「論理の反転（NOT）」 を意味する。
```
~data['国語'].duplicated()
```

```
始まり : 終わり : ステップ
```

## one-hot表現とは
一つだけが1（hot）であとは、0（cold）という表現方法。

例: 色が ["赤", "青", "緑"] という3カテゴリの場合
| 色 | One-hot ベクトル |
| - | ------------ |
| 赤 | `[1, 0, 0]`  |
| 青 | `[0, 1, 0]`  |
| 緑 | `[0, 0, 1]`  |

```
numpy.eye(n)
```
により、one-hot表現の行列を作ることができる。

なぜeye（目）？<br>
=> 数学で単位行列を「I（アイ）」と記号で表現するから

## min-maxスケーリング
最大値1,最小値0のデータを作る（正規化）

```
最大値1,最小値0のデータ = (既存のデータ - 最小値) / (最大値 - 最小値)
```

「既存のデータ - 最小値」で０を作る。<br>
この時の最大値は「既存のデータ[最大値] - 既存のデータ[最小値]」となる。<br>
そこで、「既存のデータ[最大値] - 既存のデータ[最小値]」を割ることで1を作ることができる。

## 行列積
行列A = m行, n列<br>
行列B = n行, p列<br>
行列A * 行列B = m行, p列

```
dataA([[a, b, c],
       [d, e, f],
       [g, h, i],
       [j, k, l],
       [m, n, o]])

×

dataB([[a', b'],
       [c', d'],
       [e', f']])

=

[
  [a*a'+b*c'+c*e', a*b'+b*d'+c*f'],
  [d*a'+e*c'+f*e', d*b'+e*d'+f*f'],
  [g*a'+h*c'+i*e', g*b'+h*d'+i*f'],
  [j*a'+k*c'+l*e', j*b'+k*d'+l*f'],
  [m*a'+n*c'+o*e', m*b'+n*d'+o*f']
]
```
